# 0040 - Seperate actions

## Goal
Give users the options to go through all actions for one tick one by one.


## Outcome
- In the UI for each selected tick it should be possible to select which action should be displayed
- It should still follow the general playthrough rule, means actions are exectued in the view, then the selected action come after it
- The play button, will not be affected
- it should be a small selection box, then the mouse hovers over the selected tick. Here the action can be then selected
